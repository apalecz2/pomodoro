# Requirements

1. Physical timer with visible time remaining. Digits: tens of minutes to single seconds = 4 digits

2. Online tracking system. Ideally from anywhere, with database for history of completed timers

3. Sync between physical timer, and online app to control from the app and provide custom times, etc.

4. Physical buttons for toggling the timer, restarting to the set amount of work time, and for the set break time

5. A way to log what was completed during a given segment -- and goals of a segment?

6. Progess bars for completion status for the number of loops I want to complete in a day?
